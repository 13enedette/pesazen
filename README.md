 Pesa Zen – Pension Management App

 Overview

Pesa Zen is a simple, senior-friendly pension management web application designed to help users easily track their pension payments, view history, and access support services.

The app focuses on clarity, accessibility, and ease of use, making it suitable for both literate and non-literate senior citizens.

 Features

*  View latest pension payment
*  Check upcoming payment dates
*  Access payment history
*  Simple reminders for important dates
*  One-tap help & support
*  Store important pension documents

---
Design Philosophy

Pesa Zen uses a **green and gold theme**:

* Green → represents stability, security, and well-being
* Gold → represents value, dignity, and financial reward

The interface is built with:

* Large text and buttons
* High contrast colors
* Minimal steps for navigation

---
Tech Stack

* Frontend: HTML, CSS
* Backend: PHP
* Data Handling: PHP (with optional MySQL for future expansion)

 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/pesa-zen.git
```

 2. Move to project folder

```bash
cd pesa-zen
```

 3. Run locally

* Place the project in your server directory (e.g., `htdocs` for XAMPP)
* Start Apache server
* Open your browser and go to:

```
http://localhost/pesa-zen/
```
